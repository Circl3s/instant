# Instant
Prefetch and instantly load your subpages, you know the deal.

## Usage
```html
<head>
  ...
  <script src="[link to Instant]"></script>
</head>
<body>
  ...
  <a is="instant-load" href="./subpage.html">Link</a>
  ...
</body>
```
